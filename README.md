# 📁 Next JS File Upload

🚀 A Full Stack File Upload Web Application built using Next.js, MongoDB, Multer, and Axios. This project allows users to upload and manage files efficiently with a modern and responsive user interface.

---

## ✨ Features

* 📤 File Upload System
* ⚡ Fast Upload Handling
* 🗂️ File Management
* 🌐 REST API Integration
* 📱 Responsive Design
* 🔒 Secure Backend Handling
* ☁️ MongoDB Database Integration

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/CheSubhro/nextjsfileupload.git
```

### Navigate to project folder

```bash
cd nextjsfileupload
```

### Install dependencies

```bash
npm install
```

### Create .env.local file

```env
MONGODB_URI=your_mongodb_connection_string
```

### Run development server

```bash
npm run dev
```

---

## 🚀 Build for Production

```bash
npm run build
npm start
```
